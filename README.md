# Sistema de Gestão de Empréstimos

Sistema desenvolvido em Python para controle de microcrédito e empréstimos com pagamento diário.

O projeto permite cadastrar clientes, registrar empréstimos com juros automáticos, acompanhar pagamentos e visualizar um dashboard financeiro com informações atualizadas.

## Funcionalidades

- Cadastro de clientes
- Registro de empréstimos
- Aplicação automática de 30% de juros
- Controle de pagamentos diários
- Acréscimo de dias em caso de atraso
- Dashboard com:
  - número de clientes cadastrados
  - quantidade de empréstimos ativos
  - valor total a receber
  - lucro total em juros

## Como funciona

Ao iniciar o sistema, um dashboard mostra automaticamente as principais informações financeiras.

Em seguida é possível acessar o menu para:

1. Cadastrar cliente
2. Registrar novo empréstimo
3. Registrar pagamento
4. Visualizar empréstimos ativos

Caso o pagamento diário não seja realizado, o sistema adiciona dois dias ao contrato.

## Tecnologias utilizadas

- Python
- Estrutura de dados (listas e dicionários)
- Lógica de controle financeiro

## Objetivo do projeto

Este projeto foi desenvolvido como exercício de lógica de programação e criação de sistemas financeiros simples em Python.

Também pode ser utilizado como base para evolução de um sistema mais completo de gestão de crédito.
